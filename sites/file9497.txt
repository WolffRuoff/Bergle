	</li>
</ol>

<p dir="ltr" style="margin-left: 80px;"><span style="font-size:14px;">examples include sites such as: <a href="http://www.nytimes.com">NY Times</a>, <a href="http://www.cnn.com">CNN</a>, <a href="http://www.usatoday.com">USA Today</a>, <a href="https://www.amazon.com">Amazon</a>, etc.</span></p>

<ol start="5">
	<li dir="ltr">
	<p dir="ltr" role="presentation"><span style="font-size:14px;"><strong><a id="Down" name="Down"></a>Enter the url in</strong>&nbsp; <a href="http://www.isitdownrightnow.com/">http://www.isitdownrightnow.com/</a> to ensure that the vendor/resource website is available</span></p>
	</li>
	<li dir="ltr">
	<p dir="ltr" role="presentation"><span style="font-size:14px;"><strong><a id="EZProxy" name="EZProxy"></a>If you are receiving an error page that includes&nbsp;&quot;</strong><span style="font-weight: 700; font-size: 14px;">EZ Proxy&quot;</span>&nbsp;</span></p>

	<ul>
		<li>
		<p dir="ltr" role="presentation"><span style="font-size:14px;">Try removing&nbsp;the&nbsp;part of the URL that says: <a href="https://muhlenberg.idm.oclc.org/login?url">https://muhlenberg.idm.oclc.org/login?url</a>=&nbsp;</span></p>
		</li>
		<li>
		<p dir="ltr" role="presentation"><span style="font-size:14px;">If the site then resolves, please report the&nbsp;problem to the library-- See&nbsp;below...</span></p>
		</li>
	</ul>
	</li>
	<li dir="ltr">
	<p dir="ltr" role="presentation"><span style="font-size:14px;"><strong><a id="Paywall" name="Paywall"></a>If you see a &ldquo;paywall&rdquo; or if content is locked or for any other errors:</strong></span></p>

	<ul>
		<li>
		<p dir="ltr" role="presentation"><span style="font-size:14px;"></span><span style="font-size:14px;"><span style="font-weight: 700; font-size: 16px;">Send for help!&nbsp; (See below)</span></span></p>
		</li>
	</ul>
	</li>
</ol>

<p><span style="font-size:14px;"><span style="font-family: arial, helvetica, sans-serif;"><span id="docs-internal-guid-fabd2f12-7fff-5eb8-fa24-904d231d1acd"><span style="color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-style: italic; font-variant-numeric: normal; font-variant-east-asian: normal; vertical-align: baseline; white-space: pre-wrap;">Use the &ldquo;Report a broken link&rdquo; functionality in the catalog,&nbsp;</span></span></span></span></p>

<p dir="ltr" style="margin-top: 0pt; margin-bottom: 0pt; line-height: 1.8; text-align: center;"><span style="font-size:14px;"><span style="font-family: arial, helvetica, sans-serif;"><span id="docs-internal-guid-fabd2f12-7fff-5eb8-fa24-904d231d1acd"><span style="color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-style: italic; font-variant-numeric: normal; font-variant-east-asian: normal; vertical-align: baseline; white-space: pre-wrap;">or send a description of the problem (with screenshots, if possible) to: </span></span></span></span></p>

<p dir="ltr" style="margin-top: 0pt; margin-bottom: 0pt; line-height: 1.8; text-align: center;"><span style="font-size:16px;"><span style="font-family: arial, helvetica, sans-serif;"><span style="color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-style: italic; font-variant-numeric: normal; font-variant-east-asian: normal; vertical-align: baseline; white-space: pre-wrap;">libraryresources [at] muhlenberg.edu</span></span></span></p>

<p dir="ltr" style="margin-top: 0pt; margin-bottom: 0pt; line-height: 1.8; text-align: center;">&nbsp;</p>

<p><span style="font-size: 16px;"><span style="font-family: arial, helvetica, sans-serif;"><span id="docs-internal-guid-fabd2f12-7fff-5eb8-fa24-904d231d1acd"><span style="color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-variant-numeric: normal; font-variant-east-asian: normal; vertical-align: baseline; white-space: pre-wrap;">Please include the following information in your report:</span></span></span></span></p>

<ul dir="ltr">
	<li style="line-height: 1.8; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 14px;"><span style="font-family: arial, helvetica, sans-serif;"><span style="color: rgb(0, 0, 0); background-color: transparent; font-weight: 700; font-variant-numeric: normal; font-variant-east-asian: normal; vertical-align: baseline; white-space: pre-wrap;">The title of the resource?</span></span></span></li>
	<li><span style="font-size: 14px;"><span style="background-color: transparent; color: rgb(0, 0, 0); font-weight: 700; white-space: pre-wrap; font-family: arial, helvetica, sans-serif;">What collection/resource is it from?</span></span></li>
	<li><span style="font-size: 14px;"><span style="background-color: transparent; color: rgb(0, 0, 0); font-weight: 700; white-space: pre-wrap; font-family: arial, helvetica, sans-serif;">What is the error message saying? (Screen shots help!)</span></span></li>
</ul>

<p><span style="font-family:arial,helvetica,sans-serif;"></span></p>

<ul dir="ltr">
	<li style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><strong><span style="font-family:arial,helvetica,sans-serif;"><span id="docs-internal-guid-fabd2f12-7fff-5eb8-fa24-904d231d1acd"><span style="font-size: 11pt; color: rgb(0, 0, 0); background-color: transparent; font-variant-numeric: normal; font-variant-east-asian: normal; vertical-align: baseline; white-space: pre-wrap;">Remember to include your name and email (for follow-up)!</span></span></span></strong></li>
</ul>
