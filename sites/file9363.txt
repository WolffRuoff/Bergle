				<p><span style="font-size:14px;"><span style="font-weight: 700;">F<span style="font-weight: 700;">or&nbsp;an effective search for a musical work, it is helpful to know</span>&nbsp;the following pieces of information:</span></span></p>

	<li><span style="font-size:14px;">The composer&rsquo;s [full] name;</span></li>
	<li><span style="font-size:14px;">The nationality of the composer;</span></li>

	<li><span style="font-size:14px;">The name of a larger work from which a piece comes;</span></li>

	<li><span style="font-size:14px;">The Key signature&nbsp;of the piece;</span></li>
	<li><span style="font-size:14px;">Instrumentation (or original instrumentation) of the work;</span></li>
	<li><span style="font-size:14px;">The score format or edition that you desire.</span></li>

							<div id="s-lg-box-collapse-25464029" >
								<div class="s-lib-box-content">
									
			<div id="s-lg-content-58684463" class="  clearfix">
				<p>Enter the composer&#39;s full name (Last name first) in Encompass Search.&nbsp;</p>

<ul>
	<li><strong>If you are sure of the correct spelling</strong>, enclose&nbsp;the name in quotes or enter the name following an author search prefix (au:) for more accuracy.</li>
	<li><strong>If you aren&#39;t sure of the spelling</strong>, enter as much of the name as you are sure of and truncate the search by adding an asterisk.&nbsp;
	<ul>
		<li>For example: Stravinsk* or Rachmanino*</li>
	</ul>
	</li>
</ul>

<p>Remember to&nbsp;<strong>consider alternate spellings</strong>&nbsp;for names that are derived from a language that uses a non-Roman alphabet&nbsp;(for example&nbsp;Russian, Hebrew, Chinese, etc.)&nbsp;&nbsp;Russian names, such as Rachmaninoff, Stravinsky, and Tchaikovsky can be especially problematic. For example:&nbsp; Pyotr Ilyich Tchaikovsky&#39;s last name&nbsp;can also be spelled as&nbsp;Chaikovsky, Chaikovskii,&nbsp;Tschaikowsky, or the fully Anglicized version: Peter Ilich Tchaikovsky.&nbsp;</p>

<p>To help with searches of this type, use&nbsp;<a href="https://libraryguides.muhlenberg.edu/usingencompass/truncation" onclick="window.open(this.href, '', 'resizable=no,status=no,location=no,toolbar=no,menubar=no,fullscreen=no,scrollbars=no,dependent=no'); return false;">TRUNCATION</a>&nbsp;to search for the variant forms of the name.</p>

<p>Also remember:&nbsp;<strong>some composers come from prolific families</strong>&nbsp;of composers.&nbsp;For example, the Bach family included&nbsp;Johann Sebastian, Johann Christian, Wilhelm Friedemann, and Carl Philipp Emanuel, among others.&nbsp;&nbsp;In this case--four composers with the same last name--it&#39;s important to know a first name or first and middle name.</p>

			<div id="s-lg-content-58685202" class="  clearfix">
				<p>This can help determine the language of your search terms.&nbsp; Any good biographical dictionary should note the composer&#39;s nationality.</p>

<p>For example, if we know that Mozart was Austrian, we know that many of his works were originally titled in German.&nbsp; Therefore it might help to search for the title using the German words.&nbsp;</p>

<p>So, when searching for a piano sonata by Mozart, we would search : &quot;Klaviersonate AND Mozart&quot;</p>

</ul>

<p>(Note that Klaviersonate (singular) and Klaviersonaten (plural) are compound words in German)</p>

<p>The&nbsp;<strong>opus number</strong>&nbsp;is the &quot;work number&quot; that is assigned to a musical composition, or to a set of compositions, to indicate the (approximate) chronological order of the composer&#39;s production.&nbsp; It is often abbreviated as &#39;op.&#39;</p>

<p>A&nbsp;<strong>thematic catalog</strong>&nbsp;(or thematic index) is a list of a&nbsp;composer&#39;s works. <p>&quot;BWV&quot; = Bach Werke Verzeichnis (J.S. Bach)<br />
&quot;D&quot; = Deutsch (Schubert)<br />
&quot;Hob.&quot; = Hoboken (F.J. Haydn)<br />
&quot;K&quot; = Kochel (Mozart)</p>

<p>&quot;RV&quot; = Ryom-Verzeichnis (Vivaldi)</p>

<p>You can often find thematic catalog numbers using the &quot;works&quot; section at the end of composer entries in The New Grove Dictionary (or Grove Music Online).</p>

	<li>vocal score = piano-vocal score (shows only a reduced score for voice accompanied by the piano)</li>
	<li>piano reduction = full score (usually an instrumental accompaniment for a solo) reduced so that it can be played on the piano.</li>
	<li>score and parts (or just parts) for solo music with piano or chamber music.</li>
</ul>

<p>For more information on score formats and editions, see the &quot;<a href="https://libraryguides.muhlenberg.edu/musicscores/editions">Music Editions</a>&quot; tab.</p>
